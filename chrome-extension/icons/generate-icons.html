Gerador de Ícones - Prizely WhatsApp Exporter

Para gerar os ícones PNG necessários para a extensão:

  1. Use uma ferramenta online de conversão SVG → PNG:
    • SVGtoPNG.com
    • CloudConvert
  2. Converta os arquivos SVG da pasta icons/:
    • icon16.svg → icon16.png
    • icon48.svg → icon48.png
    • icon128.svg → icon128.png
  3. Ou use o script Node.js: node icons/convert-icons.js (requer npm install canvas)

Método Alternativo: Screenshot dos SVGs

Você também pode abrir cada arquivo SVG no navegador e fazer screenshot:

  1. Abra icon16.svg no navegador
  2. Faça screenshot do ícone
  3. Salve como PNG com o tamanho correto
  4. Repita para os outros tamanhos
